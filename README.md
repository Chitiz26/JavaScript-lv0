# JavaScript Level 0
10 JavaScript exercises for beginners.

## Questions

1. `question1.js` - Data Types & Variables
2. `question2.js` - Operators
3. `question3.js` - Type Coercion
4. `question4.js` - If-Else
5. `question5.js` - Loops
6. `question6.js` - Functions
7. `question7.js` - Arrays
8. `question8.js` - Objects
9. `question9.js` - Hoisting
10. `question10.js` - Basic DOM

## How to Run

Run with Node.js:
```bash
node question1.js
```

For DOM (question10.js), use an HTML file with:
```html
<div id="myDiv"></div>
<button id="myButton"></button>
<script src="question10.js"></script>
```
